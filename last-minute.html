Last Minute Hotel Booking Portal

Project Brief

A hospitality customer wanted to start hotel booking marketplace specially dedicated for last minute deals. Their goal was to enable people to find and book last minute great deals on hotels.

Problem Statement

A end-end solution for hotel booking with all device support and infrastructure capacity to handle large traffic. To allow hoteliers to add/update rates/inventory for their hotels for 3 years in advance. Allow corporates HR to analyse travel cost of all employees on single dashboard and avoid individual employees the overhead to submit stay bills and thus more transparency.

Solution

Smaera proposed a hotel booking portal hosted on Amazon Web Services (AWS).

Application Development – Smaera illustrated the benefits of agile development of hotel booking engine with cloud DevOps, unified collaboration, and delivered faster while keeping everything safe.

The key benefits included:

  • Time-to-market reduction: 40%
  • Cycle time reduction: 60%
  • Faster deliver: 40%.

    • Fast development iterations provide the freedom to innovate because teams can test more features. This reduces the risk of each tested feature. Jenkins is an open-source continuous integration (CI) tool that helps orchestrate the development processes (build, test, and deployment) with automation was used to facilitate the entire workflow.

    Cloud Security -To provide secure booking portal AWS WAF solution was used, which protect unauthorized access, use, alteration, complex attacks, keeping the site safe and online. The Site is being routed through AWS Route53 and SSL Certificates are being installed to server traffic via HTTPS.
